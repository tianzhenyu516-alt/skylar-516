# Skylar 516 - 智能检测系统

这是一个基于 STM32 的嵌入式智能检测系统，包含称重传感器、OLED 显示、舵机控制和机器视觉识别功能。

---

## 项目结构

```
skylar-516/
├── STM32/                # STM32 嵌入式程序
│   ├── Core/            # 核心架构模块 (升级版)
│   │   ├── config.h     # 系统配置文件
│   │   ├── error.h/c    # 错误处理模块
│   │   ├── state_machine.h/c  # 状态机模块
│   │   └── data_manager.h/c   # 数据管理模块
│   ├── Drivers/         # 硬件驱动
│   ├── System/          # 系统层
│   └── User/            # 用户应用层
│       └── main.c       # 主程序 v2.0
│
└── Python/              # Python 视觉识别程序
    ├── screen_test.py   # 屏幕测试
    ├── yellow_leaf_detect.py  # 黄叶检测
    └── ...
```

## 技术特点 (v2.0 升级)

### 1. 状态机架构
- 7种系统状态：初始化/校准/空闲/传感器采集/舵机触发/电机停止/错误
- 事件驱动的状态转换
- 高可扩展性的架构设计

### 2. 统一错误处理机制
- 完整错误码定义（系统/硬件/通信/数据/状态）
- 错误信息记录与上报
- 容错机制设计

### 3. 数据采集与管理
- 环形缓冲区存储历史数据
- 平均滤波与数据有效性验证
- 零点自动校准

### 4. 模块化软件架构
```
┌─────────────────────────────────────────┐
│          应用层 (User)                   │
├─────────────────────────────────────────┤
│      核心层 (Core)                       │
│  ┌──────┐ ┌──────┐ ┌────────┐ ┌─────┐  │
│  │状态机│ │错误管│ │数据管理│ │配置│  │
│  └──────┘ └──────┘ └────────┘ └─────┘  │
├─────────────────────────────────────────┤
│     驱动层 (Drivers)                     │
│  OLED, HX711, Servo, Relay, Signal      │
├─────────────────────────────────────────┤
│     硬件层 (STM32F10x)                   │
└─────────────────────────────────────────┘
```

## 硬件清单

### 主控
- STM32F103C8T6 / STM32F103RC

### 外设
- HX711 称重传感器
- 0.96寸 OLED 显示屏 (I2C)
- 舵机 x2
- 继电器模块
- 按键/信号输入

---

## 功能说明

### STM32 程序功能
1. **称重检测**
   - HX711 传感器数据采集
   - 5次采样平均滤波
   - 重量阈值触发（10g）

2. **OLED 显示**
   - 实时重量显示
   - 阈值显示
   - GPIO 状态显示

3. **舵机控制**
   - 舵机1：单步推物动作（0-270度）
   - 舵机2：循环打印动作（0-180度）

4. **电机控制**
   - PA5 信号检测
   - 自动停机6秒后恢复
   - 消抖处理

### Python 程序功能
1. 黄叶检测算法
2. 屏幕测试脚本

## 快速开始

### STM32 开发
1. 使用 Keil MDK 打开项目
2. 配置硬件（确保引脚连接正确）
3. 编译并烧录到 STM32
4. 开机自动校准，稳定后运行

### Python 开发
```bash
cd Python
pip install opencv-python numpy
python yellow_leaf_detect.py
```

## 引脚定义

### HX711
- SCK -> PB0
- DT -> PB1

### OLED
- SCL/SDA (I2C)

### 舵机
- 舵机1 -> PWM PA1
- 舵机2 -> PWM PA11

### 继电器
- 控制引脚 (默认高电平启动)

### 信号输入
- PA5 (检测信号)
- PA4 (状态显示)
