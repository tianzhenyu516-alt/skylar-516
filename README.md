# 小青菜机创 - 智能检测系统

这是一个基于 STM32 的嵌入式智能检测系统，包含称重传感器、OLED 显示、舵机控制和机器视觉识别功能。

---

## 📦 项目简介

小青菜机创是一个综合性的嵌入式学习项目，融合了传感器采集、运动控制、显示交互和计算机视觉等技术。

**硬件平台**：STM32F103  
**开发环境**：Keil MDK  
**编程语言**：C / Python

## 🚀 技术特点 (v2.0)

### 1. 状态机架构
- **7种系统状态**：初始化、校准、空闲、传感器采集、舵机触发、电机停止、错误
- **事件驱动设计**：重量触发、信号触发、错误发生
- **状态转换清晰**：可扩展的架构设计

### 2. 统一错误处理机制
- **完整错误码定义**
  - 系统错误：初始化失败、超时
  - 硬件错误：OLED、HX711、舵机、继电器
  - 通信错误：UART、I2C、SPI
  - 数据错误：无效数据、溢出、损坏
- **错误信息记录与管理**
- **容错处理设计**

### 3. 数据采集与管理
- **环形缓冲区**：存储历史数据
- **5次采样平均滤波**
- **零点自动校准**
- **重量回差处理**：10g阈值 + 5g回差

### 4. 模块化软件架构
```
┌─────────────────────────────────────────┐
│          应用层 (User)                   │
├─────────────────────────────────────────┤
│      核心层 (Core)                       │
│  ┌──────┐ ┌──────┐ ┌────────┐ ┌─────┐  │
│  │状态机│ │错误管│ │数据管理│ │配置│  │
│  └──────┘ └──────┘ └────────┘ └─────┘  │
├─────────────────────────────────────────┤
│     驱动层 (Drivers)                     │
│  OLED, HX711, Servo, Relay, Signal      │
├─────────────────────────────────────────┤
│     硬件层 (STM32F10x)                   │
└─────────────────────────────────────────┘
```

## ⚡ 功能说明

### STM32 程序功能

#### 1. 称重检测
- HX711 传感器数据采集
- 5次采样平均滤波
- 重量阈值触发（默认 10g）
- 回差防止误触发（5g）

#### 2. OLED 显示
- 实时重量显示（5位数字）
- 阈值显示
- PA4 GPIO 状态显示
- 系统运行状态

#### 3. 舵机控制
- **舵机1 (PA1)**：单步推物动作
  - 0度 -> 270度（推）
  - 延时2秒
  - 270度 -> 0度（拉）
- **舵机2 (PA11)**：循环打印动作
  - 每5秒循环一次
  - 0度 -> 180度 -> 0度

#### 4. 电机控制
- **PA5 信号检测**：触发停止
- **自动延时6秒**后恢复
- **50ms消抖处理**

### Python 程序功能

#### 1. 黄叶检测算法
- OpenCV 图像处理
- 颜色阈值分割
- 轮廓检测与分析

#### 2. 屏幕测试脚本
- 简单的 UI 测试
- 信号验证

## 📝 引脚定义

### HX711 称重传感器
| 引脚 | 功能 |
|------|------|
| PB0 | SCK (时钟) |
| PB1 | DT (数据) |

### OLED 显示屏
| 引脚 | 功能 |
|------|------|
| I2C_SCL | SCL |
| I2C_SDA | SDA |

### 舵机
| 引脚 | 功能 |
|------|------|
| PA1 | 舵机1 PWM (推物) |
| PA11 | 舵机2 PWM (打印) |

### 继电器
| 引脚 | 功能 |
|------|------|
| (配置中) | 继电器控制 |

### 信号输入
| 引脚 | 功能 |
|------|------|
| PA5 | 停止触发信号 |
| PA4 | 状态输出显示 |

## 🎯 快速开始

### STM32 开发

1. **打开项目**：使用 Keil MDK 打开项目文件夹
2. **硬件连接**：按照引脚定义连接硬件
3. **编译烧录**：配置调试器（JLink / STLink），编译并下载
4. **自动校准**：上电后自动执行零点校准
5. **正常运行**：稳定后进入工作状态

### Python 开发

```bash
# 进入 Python 目录
cd Python

# 安装依赖
pip install opencv-python numpy

# 运行黄叶检测
python yellow_leaf_detect.py

# 运行屏幕测试
python screen_test.py
```

## 🔧 系统配置

所有配置参数集中在 `STM32/Core/config.h`：

```c
// 系统配置
#define SYSTEM_TICK_INTERVAL    10      // 主循环频率 (ms)

// 重量配置
#define HX711_WEIGHT_THRESHOLD  10.0f   // 触发阈值 (g)
#define HX711_HYSTERESIS        5.0f    // 回差 (g)

// 舵机配置
#define SERVO1_DELAY_MS         2000    // 推物动作延时
#define SERVO2_LOOP_INTERVAL    5000    // 打印动作周期

// 信号配置
#define SIGNAL_DEBOUNCE_MS      50      // 消抖时间
#define SIGNAL_STOP_DURATION_MS 6000    // 停止时长
```
